This is a Flight Tracker app that allows users to track flights in real-time using the Adsb.lol API.

On the main page, users can view a map with all planes within 250 nautical mile radius of their device. When a plane is clicked, a card opens displaying an image or a placeholder of the plane type, along with details about the flight and the aircraft, including dynamic real-time data.

The app was built using the Ionic Framework and Angular, providing a seamless and responsive user experience.

It is designed to work flawlessly in both landscape and horizontal views, and offers two themes - light and dark - based on the user's phone settings.

The map functionality is powered by the Apple Mapkit JS API, utilizing the latest updates including advanced markers. Additionally, icons and images have been customized using Photoshop and Inkscape to align with the app's theme.

I made the code available on Github, you can reach it by clicking on the github icon in the main page top left corner.
